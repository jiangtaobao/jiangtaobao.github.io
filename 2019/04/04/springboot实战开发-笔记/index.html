springboot实战开发-笔记

2019-04-04

一.核心
1.基本配置：
@SpringBootApplication@SpringBootApplication(exclude = {关闭特定的自动配置.class})是组合注解：
@Configuratioon、@EnableAutoConfiguration、@ComponetScan
自动扫描同级包及其子包
2.外部配置：
@ImportResource({“classpath:xxx.xml”,””…})
@ConfigurationProperties(prefix = “”,locations = “”)
类型安全配置基于配置文件
3.日志：
默认Logback作为日志框架
4.Profile配置
application-{profile}.properties/yml
–spring.profiles.active = {profile}
二.运行原理
EnableAutoConfigurationImportSelector使用SpringFactoriesLoad.loadFactoryNames方法来扫描具有META-INF/spring.factories文件的jar包
三.Web开发

@{} —— 引用静态资源
